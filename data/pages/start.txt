====== Control de Acceso y Prenómina (CAP) 1.0 ======
====== Manual de Usuario ======
===== Introducción =====
CAP es una aplicación web, desarrollada para la agilización del proceso de generación de prenóminas, basandose en información generada por el reloj checador incluido en CAP (CAPRelojChecador).\\
CAP esta compuesto por **CAP Administrador** y **CAP Reloj Checador**.\\
El siguiente documento es una guía para el uso de estos sistemas.\\

===== Preguntas frecuentes =====
  * ¿Cómo se da de alta un nuevo empleado? - [[wiki:altaempleado | Alta de empleado ]]
  * ¿Cómo se asigna el horario a un empleado? - [[wiki:asignarhorario | Asignar horario ]]
  * ¿Cómo se asigna puesto y/o departamento a un empleado? - [[wiki:asignarpuestoydetpo | Asignar puesto y depto ]]
  * ¿Cómo se indica que un empleado estará de descanso, en día festivo o en capacitación? - [[wiki:nolaborables | Días no laborables ]]
  * ¿Cómo se pueden justificar o indicar faltas, retardos, horas extra, falta de checadas? - [[wiki:ajustesprenomina | Ajustes para prenómina ]]

===== Roles de usuario =====
El sistema esta hecho para ser utilizado por diferentes perfiles de usuarios (roles), a continuación se muestra una lista de los roles y funciones a las cuales tendrán acceso para navegar dentro del manual y encontrar fácilmente los temas de interés.
===Administrador de sistema===
Este perfil tendrá acceso principalmente a las vistas de administración, para una configuración del sistema.
  * Creación, modificación y eliminación de usuarios - [[wiki:user | Usuarios]].
  * Creación y ordenamiento de menús - [[wiki:menu | Menú]].
  * Creación, modificación y eliminación de roles - [[wiki:rol | Rol]].
  * Colocar que menús se pueden ver por cada rol - [[wiki:menurol | Menú vs. rol]].
  * Asignar roles a los usuarios - [[wiki:userrol | Usurio vs. rol]].
  * Consulta de empleados con huella digital y faltantes, y poder modificar si pueden checar con huella o código o ambos - [[wiki:huellas | Huellas digitales]].
